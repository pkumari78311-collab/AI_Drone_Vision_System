# AI Drone Vision System 🚁🤖

An AI-powered autonomous drone navigation system that combines YOLOv8 object detection, OpenCV computer vision, and AirSim drone simulation to create a real-time perception → decision → action pipeline.

---

## 📌 Project Overview

This project demonstrates how Artificial Intelligence can be used to enable autonomous drone behavior. The system detects objects in real time using a webcam, identifies their position within the frame, makes navigation decisions, and interacts with a simulated drone environment using Microsoft's AirSim.

The project was developed as a practical application of:

- Computer Vision
- Deep Learning
- Object Detection
- Autonomous Navigation
- AI Decision Making

---
